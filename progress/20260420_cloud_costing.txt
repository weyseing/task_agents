# Cloud Costing Plan — Task Agents (2026-04-20)

Target: small company, low budget, deploy this repo as-is.
Decision: Google Cloud Run backend + Cloudflare frontend stack.

================================================================
STACK
================================================================

1. Frontend            Cloudflare Pages         (React/Vite static)
2. Backend             Google Cloud Run         (FastAPI + SSE)
3. Database            Neon Postgres            (managed serverless)
4. Object Storage      Cloudflare R2            (S3-compatible)
5. DNS / TLS / CDN     Cloudflare               (proxies Cloud Run)
6. LLM API             OpenAI / Anthropic       (via LiteLLM)

================================================================
FREE TIER LIMITS & OVERAGE
================================================================

--- Cloudflare Pages ---
Free:
  - Bandwidth: unlimited
  - Requests: unlimited
  - Sites: unlimited
  - Builds: 500 / month
  - Concurrent builds: 1
  - Files per deploy: 20,000 (25 MB max each)
Overage: $5/mo Pro = 5,000 builds, 5 concurrent

--- Google Cloud Run ---
Free (per month, permanent):
  - 2,000,000 requests
  - 180,000 vCPU-seconds
  - 360,000 GB-seconds memory
  - 1 GB egress (N. America)
Overage:
  - Requests:  $0.40 / 1M
  - vCPU:      $0.000024 / vCPU-sec
  - Memory:    $0.0000025 / GB-sec
  - Egress:    $0.12 / GB
Hard limits:
  - Max request timeout: 60 min
  - Max memory: 32 GiB, Max CPU: 8 vCPU
Recommended spec: 1 vCPU, 1 GiB, concurrency 20, min=0, max=5

--- Neon Postgres (Free tier) ---
Free:
  - Storage: 0.5 GB
  - Compute: 0.25 vCPU, 1 GB RAM (autoscale, auto-suspend)
  - Compute hours: 191.9 hrs/mo
  - Projects: 10, Branches: 10 per project
  - PITR: 24 hours
  - Data transfer: 5 GB/mo
Overage tiers:
  - Launch:   $19/mo  — 10 GB storage, always-on, 7-day PITR
  - Scale:    $69/mo  — 50 GB storage, 14-day PITR
  - Business: $700/mo — 500 GB storage

--- Cloudflare R2 ---
Free (per month):
  - Storage: 10 GB
  - Class A ops (writes): 1M
  - Class B ops (reads): 10M
  - Egress: UNLIMITED, FREE
Overage:
  - Storage: $0.015 / GB / mo
  - Class A: $4.50 / 1M
  - Class B: $0.36 / 1M
  - Egress:  $0 (still free)

--- Cloudflare DNS / TLS ---
Free:
  - Unlimited DNS records + queries
  - Universal SSL auto-renewed
  - DDoS protection (L3/L4/L7)
  - 5 firewall rules, 3 page rules
Overage: Pro $20/mo for WAF, more rules
Domain registration: at-cost (~$10/year .com)

--- LLM API (pay-per-token, no free tier) ---
OpenAI:
  - GPT-4o-mini:  $0.15 in / $0.60 out   per 1M tokens
  - GPT-4o:       $2.50 in / $10.00 out
  - GPT-4.1:      $3.00 in / $15.00 out
Anthropic:
  - Claude Haiku 4.5:  $1.00 in / $5.00 out
  - Claude Sonnet 4.6: $3.00 in / $15.00 out
  - Claude Opus 4.7:   $15.00 in / $75.00 out
Optimization: prompt caching (50-90% discount on cached input)
Free alt: Google Gemini free tier (rate-limited) via LiteLLM

================================================================
KEY NOTES
================================================================

- Infra caps out ~$150/mo even at significant scale; LLM cost dominates.
- Prompt caching is the biggest cost lever — enable on OpenAI/Anthropic.
- Keep Cloud Run + Neon in same region to avoid egress charges.
- Serve files via R2 (free egress), NOT through Cloud Run ($0.12/GB).
- Scale to zero by default (min=0). Add min=1 (+$6-10/mo) only if cold
  starts bother users.
- 60-min request timeout on Cloud Run fits long agent loops.

================================================================

  --memory 1Gi \
  --cpu 1 \
  --concurrency 20 \
  --timeout 3600 \
  --min-instances 0 \
  --max-instances 5 \
  --allow-unauthenticated
