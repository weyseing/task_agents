# Cloud Costing Plan — Task Agents (2026-04-20)

Target: small company, low budget, deploy this repo as-is (Python FastAPI + React).

================================================================
INFRASTRUCTURE STACK
================================================================

1. Frontend        Cloudflare Pages        (React/Vite static)
2. Backend         Google Cloud Run        (FastAPI + SSE, Python)
3. Database        Neon Postgres           (managed serverless)
4. Object Storage  Cloudflare R2           (S3-compatible, free egress)
5. DNS/TLS/CDN     Cloudflare              (proxies Cloud Run)
6. LLM API         See LLM strategy below

================================================================
FREE TIER LIMITS & OVERAGE
================================================================

Cloudflare Pages
  Free: unlimited bandwidth + requests, 500 builds/mo, unlimited sites
  Overage: $5/mo Pro (5,000 builds)

Google Cloud Run
  Free (monthly): 2M requests, 180k vCPU-sec, 360k GB-sec, 1 GB egress
  Overage: $0.40/1M req, $0.000024/vCPU-sec, $0.12/GB egress
  Hard limits: 60-min request timeout, 32 GiB RAM, 8 vCPU
  Recommended: 1 vCPU, 1 GiB, concurrency=20, min=0, max=5

Neon Postgres
  Free: 0.5 GB storage, auto-suspend, 10 branches, 24hr PITR
  Overage: Launch $19/mo (10GB), Scale $69/mo (50GB)

Cloudflare R2
  Free: 10 GB storage, 1M writes, 10M reads, UNLIMITED free egress
  Overage: $0.015/GB storage, egress stays free

Cloudflare DNS/TLS
  Free: unlimited queries, Universal SSL, DDoS protection
  Domain: ~$10/year (at-cost)

================================================================
OLLAMA SELF-HOST VERDICT
================================================================

NOT cheaper for small low-budget company. API wins.

  GPU VPS rental (always-on 24GB):    $180–430/mo  (15–30x API cost)
  Mac Studio M4 Max 128GB (owned):    $4,000 upfront + $15 electricity
                                       = never pays back at small scale
  Existing hardware reuse:             ~$0 marginal, but weaker quality

Quality ceiling self-hosted: Llama 4 70B / Qwen 3 72B tier
  — BELOW DeepSeek V4 API
  — Cannot run Claude Opus 4.7, GPT-5.4, Gemini 3.1 Pro, DeepSeek V4 locally

When self-hosting makes sense:
  - Privacy-critical (air-gapped, regulated data)
  - Already own unused GPU hardware
  - API spend > $300/mo (not this scale)

Use Ollama for DEV/TEST only (existing cli/ollama/ scripts).

================================================================
KEY OPTIMIZATION RULES
================================================================

1. Enable prompt caching on all paid tiers (30–50% savings, zero effort)
2. Keep Cloud Run + Neon in same region (avoid egress charges)
3. Serve files via R2 not Cloud Run (free egress)
4. Scale to zero (min=0); pay $6–10/mo min=1 only if cold starts bother users
5. Cap max_tokens on output (Opus especially)
6. Truncate old conversation turns; summarize rest
7. Route by task complexity — use Cerebras free primary, escalate when needed

================================================================
DEPLOY COMMANDS
================================================================

Backend (Cloud Run):
  gcloud run deploy task-agents-backend \
    --source ./backend \
    --region us-central1 \
    --memory 1Gi --cpu 1 \
    --concurrency 20 --timeout 3600 \
    --min-instances 0 --max-instances 5 \
    --allow-unauthenticated

Frontend (Cloudflare Pages):
  - Push repo to GitHub
  - Cloudflare dashboard → Pages → Connect repo
  - Build: cd frontend && npm install && npm run build
  - Output: frontend/dist
